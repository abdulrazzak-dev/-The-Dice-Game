# 🎲 The Dice Game

A classic, interactive 2-player dice game built with HTML, CSS, and vanilla JavaScript. This project was developed as part of a "Boss Level Challenge" to master DOM manipulation, random number generation, and responsive web design.

## 🚀 Live Demo
Check out the live game here: [https://abdulrazzak-dev.github.io/-The-Dice-Game/](https://abdulrazzak-dev.github.io/-The-Dice-Game/)

## 🛠️ How It Works
* **Randomization:** Every time the page is refreshed, the game generates two random numbers between 1 and 6.
* **Dynamic UI:** The site automatically updates the dice images to match the generated numbers.
* **Win Logic:** The application compares the two results and dynamically updates the page header to announce the winner or declare a draw.

## 💻 Tech Stack
* **HTML5:** Structuring the game layout.
* **CSS3:** Styling the dice interface and ensuring responsiveness.
* **JavaScript:** Handling the game logic, random dice generation, and DOM manipulation to update the UI in real-time.
